Git 仓库迁移

发表于 2017-11-06 更新于 2020-08-04 分类于 Git 阅读次数： Valine：

最近公司重新规划了服务器，由于不想损失之前的 git 仓库，commit 等信息，所以需要对 git 仓库进行迁移。

经过尝试，记录如下。

有两个目录

  1. 原来的 git 仓库，old.git
  2. 新的 git 仓库，new.git

迁移步骤

  1. 在新服务器新建 git bare 仓库

  2. 本地克隆旧的 git 仓库

  3. 最后推送到新的 git 仓库
